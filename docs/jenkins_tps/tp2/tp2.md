TP2 : Création et compilation d'un projet Maven
Objectif : industrialiser le build avec Maven et publier les résultats de tests.
1
Installer le plugin « Maven Integration » puis déclarer Maven dans Global 
Tool Configuration.
2
Créer un job de type « Projet Maven ».
3
Pointer vers le pom.xml et définir les goals : clean install.
4
Lancer le build : compilation, exécution des tests, packaging (jar/war).
5
Vérifier la publication automatique du rapport JUnit (Surefire) et la 
tendance.
